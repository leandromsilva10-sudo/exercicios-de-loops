# Exercícios de Loops

Este repositório contém uma coleção de exercícios práticos focados em estruturas de repetição (loops) em programação.

## Objetivo

Fortalecer o entendimento e a prática de diferentes tipos de loops:
- `for`
- `while`
- `do...while`
- `for...each`
- Loops aninhados

## Conteúdo

Os exercícios estão organizados por nível de dificuldade:

- **Básico**: Introdução aos loops e conceitos fundamentais
- **Intermediário**: Loops aninhados e padrões mais complexos
- **Avançado**: Otimizações e casos de uso reais

## Como Usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/leandromsilva10-sudo/exercicios-de-loops.git
   cd exercicios-de-loops
   ```

2. Navegue até o exercício desejado

3. Resolva o exercício seguindo as instruções fornecidas

4. Teste sua solução
